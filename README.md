# Hexo Theme Uno

> A lite theme for Hexo. | 极简美。

## Chinese

### 特性

Hexo Theme Uno 足够轻，具有极简美，只包含 Blog 最基本的元素。

### 使用

您可以下载主题的源码，并将其拷贝至 `themes/uno` 文件夹下。在**博客设置(`_config.yml`)**中启用：
```yaml
theme: "uno"
```

当然，我们更推荐您使用 `npm` 安装:
```bash
npm install --save hexo-theme-uno
```

在**博客设置(`_config.yml`)**中启用：
```yaml
theme: "uno"
```
